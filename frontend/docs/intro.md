# Introduction

Welcome to the **Physical AI & Humanoid Robotics Book** - a comprehensive guide to mastering ROS 2, physics simulation, and digital twin development for humanoid robots.

## About This Book

This book teaches you to build complete robotic systems from the ground up, combining theoretical knowledge with hands-on practical experience. You'll learn to create robotic nervous systems with ROS 2, simulate physical environments with Gazebo, and build high-fidelity digital twins with Unity.

## What You'll Learn

- **ROS 2 Fundamentals**: Master nodes, topics, services, and actions
- **Physics Simulation**: Create realistic simulations with Gazebo
- **Sensor Integration**: Work with LiDAR, cameras, and IMUs
- **Digital Twins**: Build real-time synchronized visualizations with Unity
- **Humanoid Robotics**: Model and control humanoid robots

## How to Use This Book

This book is organized into **progressive modules** that build on each other:

1. **[Module 1: The Robotic Nervous System (ROS 2)](/docs/module-1-ros2-nervous-system/overview)**
   - Learn ROS 2 fundamentals and architecture
   - Write Python ROS 2 nodes
   - Model humanoid robots with URDF

2. **[Module 2: The Digital Twin (Gazebo & Unity)](/docs/module-02-digital-twin/overview)**
   - Physics simulation with Gazebo
   - Sensor integration and visualization
   - Real-time digital twins with Unity

## Prerequisites

Before starting, ensure you have:

- **Operating System**: Ubuntu 20.04/22.04 LTS or Windows with WSL2
- **Python**: Version 3.8 or higher
- **Programming**: Basic familiarity with Python and Linux command line
- **Hardware**: Computer with at least 8GB RAM (16GB recommended for Unity)

## Getting Started

Ready to begin? Start with **[Module 1: The Robotic Nervous System](/docs/module-1-ros2-nervous-system/overview)** to build your foundation in ROS 2!
