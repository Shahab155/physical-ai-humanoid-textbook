# Module 1: The Robotic Nervous System (ROS 2)

## Overview

This module introduces the fundamental concepts of ROS 2 (Robot Operating System 2) for controlling humanoid robots. You'll learn how ROS 2 provides the "nervous system" that enables robots to perceive, think, and act.

## What You'll Learn

- **ROS 2 Fundamentals**: Nodes, topics, publishers, subscribers, and the ROS 2 architecture
- **Python Agent Integration**: Writing Python scripts to control humanoid robots
- **URDF Modeling**: Creating Unified Robot Description Format files for humanoid robots

## Module Structure

This module is divided into three progressive chapters:

1. **[ROS 2 Fundamentals](./chapters/chapter-1-ros2-fundamentals.md)** - Understand the ROS 2 architecture, communication patterns, and core concepts
2. **[Python Agent Integration](./chapters/chapter-2-python-agent-integration.md)** - Learn to write Python scripts that interact with ROS 2
3. **[Humanoid URDF](./chapters/chapter-3-humanoid-urdf.md)** - Model humanoid robots using URDF

## Prerequisites

- Basic Python programming knowledge
- Familiarity with Linux command line
- Understanding of object-oriented programming concepts

## Estimated Time

- **Chapter 1 (ROS 2 Fundamentals)**: 2-3 hours
- **Chapter 2 (Python Agent Integration)**: 3-4 hours
- **Chapter 3 (Humanoid URDF)**: 2-3 hours
- **Total**: 7-10 hours

## Key Concepts

By the end of this module, you will understand:

- ✅ How ROS 2 nodes communicate via topics
- ✅ How to write Python ROS 2 nodes
- ✅ How to model robots using URDF
- ✅ How to launch and manage ROS 2 systems
